# 进入电协后基本要求

## 总体要求

分为几个方面：

1. ### 学习的态度

   具体表现能否持续的学习新的东西，而不是仅仅满足于当前会的

2. ### 学习的方法和效率

   1.形成自己的一套高效率的学习方法，（资源检索，理解，整理）

   2.需要对知识有一定深度的探索而不是仅仅会用，但是要有取舍，不能什么东西都花上很大的精力来弄，很容易捡了芝麻，丢了西瓜

   3.善用工具，学习更高效率的工具，

3. ### 学习交流，合作

   加强学习交流，这一环是大家很容易忽视的一环，但也是相对来说最为重要的一环

   不同领域的同学，要学会互相帮助，学习，

   #### 比如：

   测控的同学要做车子，那么硬件部分就可以和功率的同学合作

   有一些比较好的想法，大家一起去做一个项目
